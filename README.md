# Bat Running

A 2D parkour game about a little bat, built with Scratch and packaged to HTML using TurboWarp.

## Project Overview

Bat Running is a simple 2D side-scrolling parkour game. The player controls a bat character, avoids obstacles, and tries to complete the game.

## Features

- 2D parkour gameplay
- Character movement and jumping
- Obstacle avoidance
- Start, game over, and restart logic
- Packaged as an HTML game using TurboWarp

## Technologies Used

- Scratch
- TurboWarp
- HTML

## How to Play

Open the `index.html` file in a browser, or play it through the GitHub Pages link.
